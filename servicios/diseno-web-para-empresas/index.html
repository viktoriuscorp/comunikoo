Diseño UX/UI profesional — Leer más

Diseñamos la experiencia de usuario y la interfaz visual de tu web corporativa:

  • Arquitectura de información: estructura de páginas y navegación intuitiva
  • Wireframes de todas las páginas para validar la estructura antes del diseño
  • Diseño visual personalizado que refleja la identidad de tu marca
  • Diseño responsive: experiencia perfecta en móvil, tablet y desktop
  • Prototipos navegables en Figma para que veas tu web antes del desarrollo
Páginas esenciales para tu empresa — Leer más

Diseñamos todas las páginas que tu empresa necesita:

  • Página de inicio: propuesta de valor, servicios destacados, social proof y CTAs
  • Páginas de servicios: contenido persuasivo optimizado para SEO y conversión
  • Sobre nosotros: historia, equipo, valores y diferenciadores
  • Casos de éxito / Portfolio: resultados reales que generan confianza
  • Blog: estrategia de contenidos para atraer tráfico orgánico
  • Contacto: formulario optimizado, mapa, teléfono, WhatsApp
  • Páginas legales: política de privacidad, aviso legal, cookies (RGPD)
Desarrollo web profesional — Leer más

Desarrollo con código limpio, rápido y mantenible:

  • WordPress como CMS principal (o Webflow, según necesidades)
  • Código optimizado para velocidad y Core Web Vitals
  • Formularios con anti-spam, validación y conexión a CRM/email
  • Certificado SSL y configuración de seguridad
  • Accesibilidad web (WCAG 2.1 nivel AA)
SEO técnico y posicionamiento básico — Leer más

Tu web nace optimizada para Google:

  • Estructura de URLs optimizada y semántica
  • Meta tags y schema markup configurados
  • Velocidad de carga < 2 segundos
  • Google Analytics 4, Google Tag Manager y Search Console configurados
  • Sitemap XML y robots.txt optimizados

Preguntas frecuentes

¿Cuánto cuesta diseñar una web para mi empresa?

Nuestros proyectos de diseño web para empresas parten desde 3.000€ para webs corporativas básicas (5-6 páginas) hasta 12.000€+ para proyectos con muchas páginas, funcionalidades avanzadas o integraciones complejas. Te damos presupuesto cerrado sin compromiso.

¿Cuánto tiempo se tarda en diseñar una web corporativa?

El plazo estándar es de 5 a 8 semanas. Una web corporativa sencilla (5-8 páginas) se completa en 5-6 semanas. Proyectos más grandes o complejos pueden requerir 8-12 semanas. Te damos un cronograma detallado antes de empezar.

¿Puedo gestionar la web yo mismo después?

Sí. Todas nuestras webs incluyen un CMS (normalmente WordPress) que te permite editar textos, imágenes, publicar noticias y gestionar contenidos sin conocimientos técnicos. Incluimos formación personalizada para tu equipo.

¿Incluís textos y fotografías?

El copywriting profesional se ofrece como servicio adicional recomendado (la mayoría de clientes lo contratan). Para fotografías, podemos coordinar sesiones profesionales o trabajar con bancos de imágenes premium. Un buen contenido es fundamental para que la web funcione.

¿La web estará optimizada para Google?

Sí. El SEO técnico está integrado en cada proyecto: estructura de URLs, velocidad de carga, schema markup, meta tags y configuración de Google Analytics y Search Console. Para estrategia SEO continua (posicionamiento de keywords, contenidos, link building), ofrecemos nuestro servicio de agencia SEO.

¿Qué pasa si no me gusta el diseño?

Nuestro proceso está diseñado para que eso no ocurra. Validamos la estructura (wireframes) antes del diseño visual, y el diseño visual incluye 2 rondas de revisiones. Nunca llegamos al desarrollo con un diseño no aprobado. Si después de las revisiones no estás satisfecho, hacemos ajustes adicionales.

¿La web se verá bien en el móvil?

Absolutamente. Diseñamos con enfoque mobile first, lo que significa que la experiencia móvil se diseña primero y luego se adapta a tablet y desktop. Tu web se verá y funcionará perfectamente en cualquier dispositivo.

¿Ofrecéis mantenimiento web después del lanzamiento?

Incluimos 3 meses de soporte post-lanzamiento gratuito para resolver dudas, ajustes y correcciones. Después, ofrecemos planes de mantenimiento desde 90€/mes con actualizaciones, backups, seguridad y soporte técnico.

¿Trabajáis con empresas de fuera de Barcelona?

Sí. Aunque nuestra sede está en Barcelona, trabajamos con empresas de toda España y también con clientes internacionales. Todo nuestro proceso funciona perfectamente de forma remota con videollamadas, herramientas colaborativas y comunicación constante.

¿Puedo ver ejemplos de webs que habéis diseñado?

Sí. Solicita una reunión y te mostramos proyectos relevantes para tu sector y tipo de negocio. Preferimos enseñarte casos similares al tuyo para que veas exactamente qué podemos hacer por ti.
